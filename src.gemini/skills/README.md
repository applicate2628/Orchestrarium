# Gemini Skills

This directory is the canonical Gemini-native expertise layer for the provider pack scaffold.

Use `skills/<name>/SKILL.md` for workflow and role-like expertise that Gemini should activate on demand.

This scaffold intentionally stays minimal:

- Gemini skills are official and stable.
- Gemini extension sub-agents exist, but remain a preview feature in the official docs.
- The baseline scaffold should prefer the stable skills model and keep future preview-agent adoption optional.
- Only a minimal sample skill is included until the Gemini line needs a fuller role catalog.
